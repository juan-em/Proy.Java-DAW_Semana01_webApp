Trabajando con Apache Tomcat

Información

El servidor de Apache Tomcatt nos ofrece muchas opciones de trabajo entre ellas tenemos.

Puede manejarse tanto páginas estáticas como páginas dinámicas.

1Este servidor es un contenedor JSP/Servlet.

Las páginas estáticas se generan utilizando HTML.

2Este servidor es un contenedor JSP/Servlet.

Las páginas dinámicas se generan utilizando Servlet y JSP.

3Este servidor es un contenedor JSP/Servlet.

Solo se puede utilizar para alojar código basado en JAVA.

4Este servidor es un contenedor JSP/Servlet.

Es el contenedor que puede administrar todo el ciclo de vida de las páginas que se generan a través del Servlet y JSP.

5 No es capaz de procesar solicitudes/respuestas.

Puede ser codificado con solo el lenguaje de JAVA.
